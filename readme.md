# 🧳 银发族智能旅行助手

专为中老年朋友设计的智能旅行规划应用，提供个性化目的地推荐、行程制定、旅行清单和游记生成服务。

## ✨ 主要功能

### 1. 🌍 智能推荐与规划
- **目的地推荐**：根据季节、健康状况、预算和兴趣偏好，推荐3-5个适合的目的地
- **行程制定**：量身定制舒缓贴心的每日行程安排

### 2. 📝 清单与导游服务
- **旅行清单**：详细的行前准备清单，按类别分组（证件、药品、衣物等）
- **分类清晰**：标注【必带】和【可选】，方便整理

### 3. 🎬 旅行游记生成
- **照片生成**：根据旅行照片和补充信息生成温馨游记
- **内容丰富**：记录旅途中的美好体验和感受

## 🚀 快速开始

### 环境要求
- Python 3.8+
- pip 或 conda 包管理器

### 安装步骤

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **配置API密钥**

方法一：使用环境变量（推荐）
```bash
# Linux/macOS
export MODELSCOPE_API_KEY="ms-your-api-key-here"

# Windows (PowerShell)
$env:MODELSCOPE_API_KEY="ms-your-api-key-here"
```

方法二：复制配置文件
```bash
cp .env.example .env
# 然后编辑 .env 文件，填入您的API密钥
```

3. **运行应用**
```bash
python travel_assistant_improved.py
```

4. **访问应用**
在浏览器中打开：http://localhost:7860

## 📖 使用说明

### Tab 1: 智能推荐与规划
1. 填写季节、健康状况、预算和兴趣偏好
2. 点击"推荐目的地"获取推荐
3. 在右侧填写目的地、时长等信息
4. 点击"制定行程"获取详细行程

### Tab 2: 清单与导游服务
1. 填写目的地、旅行时长和特殊需求
2. 点击"生成清单"获取行前准备清单

### Tab 3: 旅行游记生成
1. 上传旅行照片（可选）
2. 填写补充信息
3. 点击"生成游记"获取温馨游记

## 🔧 配置说明

### API配置
- `MODELSCOPE_API_KEY`：ModelScope API密钥（必填）
- `BASE_URL`：API基础URL（可选，默认：https://api-inference.modelscope.cn/v1/）
- `MODEL_NAME`：模型名称（可选，默认：MiniMax/MiniMax-M2）

### 应用配置
- `PORT`：应用端口（可选，默认：7860）

## ⚠️ 注意事项

1. **API密钥安全**：请妥善保管您的API密钥，不要将其提交到版本控制系统
2. **网络连接**：应用需要稳定的网络连接访问API
3. **生成内容**：AI生成内容仅供参考，实际行程请结合自身情况调整
4. **健康出行**：建议出行前咨询医生，携带必要药品

## 🏥 健康建议

- 出行前进行全面体检
- 携带常用药品和处方
- 了解目的地医疗资源
- 购买旅游保险
- 告知家人旅行计划
